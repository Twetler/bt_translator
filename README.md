# BT_Translator

This is the main repository for the BT_TRANSLATOR

## Objective
The goal is to have an application that:
    * Receives the input of a headphone to start talking.
    * Allows the user to say a specific word and they hear it back its translation.

## Project Steps
### 1.1 Setup BT Connection
Allow the interface to be connected and setup the env

### 1.2 Voice to text
Find voice to text packages. We will inittially focus on listening from Dutch-Eng

### 1.3 Dictionary App
Find out a way to take from the understood word to the dictionary and format it nicely.

### 1.4 Voice back
Find voice packages that will repeat the definition back to the user
